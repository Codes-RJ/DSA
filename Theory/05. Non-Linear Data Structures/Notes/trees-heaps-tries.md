# Trees, Heaps, and Tries

## Trees

Important terms:

- root
- parent
- child
- leaf
- height
- depth

Important traversals:

- preorder
- inorder
- postorder
- level order

## Heap

- complete binary tree
- min-heap returns smallest element quickly
- max-heap returns largest element quickly
- common in priority queue problems

## Trie

- stores strings by character path
- useful for prefix search
- common in dictionary and autocomplete style questions
