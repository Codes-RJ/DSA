# Non-Linear Data Structures

Non-linear structures represent hierarchical or network relationships.

## Trees

- root, parent, child, sibling, leaf, height, depth
- binary tree, BST, balanced BST
- traversals: preorder, inorder, postorder, level order

## Heap

- complete binary tree with heap property
- min-heap and max-heap
- used in priority-based problems

## Trie

- stores strings character by character
- useful for prefix search and dictionaries

## Graph

- vertices and edges
- directed and undirected
- weighted and unweighted
- sparse and dense graphs
- representations: adjacency list and matrix

## Disjoint Set Union

- union by rank / size
- path compression
- useful in connectivity and minimum spanning tree

## Key Focus Areas

- traversals
- insert and delete
- balancing idea
- connectivity
- shortest path and spanning concepts
