# Theory Folder

This folder contains topic-wise theory notes for DSA. The content is split into separate files so it can be expanded without turning one document into an unreadable dump.

## Order of Study

1. Learning Basics
2. Complexity and Terms
3. Patterns and Problem Solving
4. Linear Data Structures
5. Non-Linear Data Structures
6. Algorithms
7. Advanced Topics

## Study Rule

For every topic, learn:

- definition
- terms
- operations
- complexity
- use cases
- common mistakes
- implementation approach
