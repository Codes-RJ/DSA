# Bit Manipulation, Segment Tree, and Fenwick Tree

## Bit Manipulation

Useful for:

- checking parity
- setting or clearing bits
- subset generation
- power of two checks

## Segment Tree

- supports range query and update efficiently
- common for sum, minimum, maximum, gcd, and custom segment data

## Fenwick Tree

- simpler than segment tree for prefix operations
- useful for prefix sum updates and queries

## When to Learn Them

After arrays, recursion, trees, and basic dynamic programming are comfortable.
