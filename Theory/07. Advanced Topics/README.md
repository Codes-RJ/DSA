# Advanced Topics

These topics are usually studied after the DSA foundation is stable.

## Advanced Structures

- segment tree
- fenwick tree
- sparse table
- monotonic stack
- monotonic queue

## Advanced Techniques

- bit manipulation
- meet in the middle
- matrix exponentiation
- advanced dynamic programming
- graph condensation concepts

## Interview-Oriented Focus

- optimize brute force ideas
- recognize reusable patterns
- write clean helper functions
- explain trade-offs clearly

## Rule for Advanced Study

Do not rush here early. Build strong basics first, then move into advanced structures and optimizations.
