# Learning Basics

Before serious DSA, programming basics must be strong.

## Topics

- variables and data types
- operators
- input and output
- conditions: `if`, `else`, `switch`
- loops: `for`, `while`, `do-while`
- functions and parameters
- arrays
- strings
- nested loops
- dry run and debugging

## Why This Matters

DSA problems are built on simple logic. Weak loop control, indexing, or condition handling usually leads to wrong answers even when the high-level idea is correct.

## Basic Skill Checklist

- print values correctly
- trace variable changes
- write loop-based solutions
- avoid off-by-one errors
- understand zero-based indexing
- pass values to functions
- return results cleanly

## Early Practice

- maximum of three numbers
- reverse a number
- count digits
- palindrome check
- factorial
- Fibonacci
- basic pattern printing
