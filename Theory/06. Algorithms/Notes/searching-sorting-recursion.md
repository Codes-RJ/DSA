# Searching, Sorting, and Recursion

## Searching

- linear search for unsorted small data
- binary search for sorted data

## Sorting

- bubble, selection, insertion: simple but slower
- merge sort: stable and `O(n log n)`
- quick sort: fast average case
- heap sort: uses heap structure

## Recursion

Recursion solves a problem by calling the same function on a smaller input.

### Must have

- base case
- shrinking problem size

### Good beginner recursive problems

- factorial
- Fibonacci
- sum of array
- reverse string
