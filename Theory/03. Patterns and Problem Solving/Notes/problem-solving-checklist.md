# Problem-Solving Checklist

Use this checklist before writing the final solution.

## Before Coding

- what is the input size?
- what output format is required?
- can I solve it by brute force first?
- does the data look sorted, hierarchical, or graph-like?
- which pattern seems to fit?

## During Coding

- am I handling empty input?
- am I using correct loop bounds?
- am I updating all variables correctly?
- does helper logic have a base case?

## After Coding

- check sample input
- test edge cases
- compute complexity
- clean variable names
- remove dead code
