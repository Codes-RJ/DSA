# Loop Patterns

Pattern printing is not just for output formatting. It trains control over rows, columns, and conditions.

## Typical Pattern Categories

- solid rectangle
- hollow rectangle
- right triangle
- mirrored triangle
- pyramid
- number triangle
- alphabet triangle

## What Pattern Questions Teach

- nested loop structure
- row and column relationship
- condition-based printing
- space handling
- symmetry and boundaries

## General Method

1. decide how many rows exist
2. find what each row prints
3. identify how spaces and symbols change
4. convert that pattern into loop variables
