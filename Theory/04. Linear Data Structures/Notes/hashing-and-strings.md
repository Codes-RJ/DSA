# Hashing and Strings

## Hashing

Hashing maps keys to positions for fast lookup.

### Typical Uses

- frequency counting
- duplicate detection
- two sum style problems
- grouping elements

### Cautions

- collisions can occur
- iteration order may not be sorted

## Strings

String problems often involve:

- frequency arrays or maps
- two pointers
- sliding window
- prefix checks
- palindrome logic

## Common Interview Patterns

- longest unique substring
- anagram detection
- character frequency comparison
- substring search
