# Pattern Cheat Sheet

## If You See This, Think About

- sorted array: binary search or two pointers
- subarray or substring: sliding window or prefix sum
- previous greater or smaller: stack
- level-based traversal: queue or BFS
- repeated subproblems: dynamic programming
- connectivity: graph traversal or DSU
- choice tree: recursion or backtracking

## Coding Habit

Do not memorize answers. Memorize how to recognize the underlying pattern.
