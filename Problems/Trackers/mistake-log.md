# Mistake Log

Track repeated mistakes here.

## Suggested Format

| Date | Problem | Topic | Mistake | Fix |
| --- | --- | --- | --- | --- |
| YYYY-MM-DD | Example Problem | Binary Search | Wrong boundary update | Recheck mid and loop condition |

## What to Record

- wrong assumption
- missed edge case
- time limit exceeded
- incorrect data structure choice
- syntax or implementation bug
