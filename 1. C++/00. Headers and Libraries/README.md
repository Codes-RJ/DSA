## What Is Covered

- `Fundamentals` covers the core headers used in DSA work.
- `Others` covers modern C++ practices and supporting topics.
- The algorithm examples in `05. Algorithms` already provide the C++ equivalents for the Java and Python source files.

## Quick Reference

Common headers used across the repository:

- `#include <iostream>` for console input and output
- `#include <vector>` for dynamic arrays
- `#include <string>` for string handling
- `#include <algorithm>` for sorting and searching
- `#include <cmath>` for math helpers
- `#include <cstdlib>` for utility routines

## How To Use

1. Start with `Fundamentals/README.md`.
2. Read `iostream`, `vector`, `string`, and `algorithm` first.
3. Move to `cmath` and `cstdlib` for utility functions.
4. Use `Others` for style and modern C++ guidance.
