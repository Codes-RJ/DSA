## Learning Order

1. Start with `iostream`.
2. Move to `vector` and `string`.
3. Learn `algorithm` for sorting and searching.
4. Use `cmath` and `cstdlib` as utility headers.

## DSA Cheat Sheet

- `iostream` for `cin`, `cout`, `cerr`, and `getline`
- `vector` for dynamic arrays and iteration
- `string` for text handling and parsing
- `algorithm` for `sort`, `find`, and binary search helpers
- `cmath` for numeric operations
- `cstdlib` for conversions, `rand`, and `system`
