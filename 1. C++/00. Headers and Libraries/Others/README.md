# Others - Additional C++ Resources

This folder contains the supporting C++ guides that currently exist in the repository.

## Available Pages

- `best_practices.md` - code style, RAII, and safe C++ habits
- `design_patterns.md` - common reusable design structures
- `debugging.md` - practical debugging workflow
- `optimization.md` - performance tuning ideas
- `memory_management.md` - ownership and resource cleanup
- `multithreading.md` - thread basics and concurrency concepts
- `templates.md` - generic programming
- `lambda_expressions.md` - inline callable objects
- `smart_pointers.md` - `unique_ptr`, `shared_ptr`, and `weak_ptr`
- `move_semantics.md` - transfer of ownership and rvalue references
- `concurrency.md` - higher-level concurrency utilities
- `filesystem.md` - file system operations
- `regular_expressions.md` - pattern matching with regex
- `miscellaneous.md` - assorted utility topics
- `stl_containers.md` - practical STL container guidance

## How To Use This Section

1. Read `best_practices.md` first.
2. Study `smart_pointers.md` next if you want modern ownership patterns.
3. Use `stl_containers.md` as a reference when choosing a container.
