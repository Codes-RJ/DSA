## 2D DP (Two-Dimensional Dynamic Programming)

### Definition

2D DP involves problems where the state is defined by two parameters. The DP table is a 2-dimensional matrix (or grid). Each cell dp[i][j] represents the answer for subproblem defined by i and j.

### General Form

dp[i][j] represents the answer for the subproblem involving:
- First i elements of one sequence and first j elements of another, OR
- Position (i, j) in a grid, OR
- Items up to i with capacity j

Transition: dp[i][j] = f(dp[i-1][j], dp[i][j-1], dp[i-1][j-1], ...)

### Classic Problems in 2D DP

### 1. Unique Paths

**Problem:** Number of ways to reach bottom-right corner of m x n grid moving only right or down.

```
State: dp[i][j] = ways to reach cell (i, j)
Base: dp[0][j] = 1 for all j, dp[i][0] = 1 for all i
Transition: dp[i][j] = dp[i-1][j] + dp[i][j-1]
Answer: dp[m-1][n-1]
```

```cpp
int uniquePaths(int m, int n) {
    vector<vector<int>> dp(m, vector<int>(n, 0));
    
    for (int i = 0; i < m; i++) {
        dp[i][0] = 1;
    }
    for (int j = 0; j < n; j++) {
        dp[0][j] = 1;
    }
    
    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            dp[i][j] = dp[i-1][j] + dp[i][j-1];
        }
    }
    
    return dp[m-1][n-1];
}
```

### 2. Minimum Path Sum

**Problem:** Find path from top-left to bottom-right with minimum sum.

```
State: dp[i][j] = minimum sum to reach cell (i, j)
Base: dp[0][0] = grid[0][0]
Transition: dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
Answer: dp[m-1][n-1]
```

```cpp
int minPathSum(vector<vector<int>>& grid) {
    int m = grid.size();
    int n = grid[0].size();
    vector<vector<int>> dp(m, vector<int>(n, 0));
    
    dp[0][0] = grid[0][0];
    
    for (int i = 1; i < m; i++) {
        dp[i][0] = dp[i-1][0] + grid[i][0];
    }
    for (int j = 1; j < n; j++) {
        dp[0][j] = dp[0][j-1] + grid[0][j];
    }
    
    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1]);
        }
    }
    
    return dp[m-1][n-1];
}
```

### 3. Longest Common Subsequence (LCS)

**Problem:** Find the length of the longest subsequence common to two strings.

```
State: dp[i][j] = LCS length of text1[0..i-1] and text2[0..j-1]
Base: dp[0][j] = 0, dp[i][0] = 0
Transition:
    if text1[i-1] == text2[j-1]:
        dp[i][j] = dp[i-1][j-1] + 1
    else:
        dp[i][j] = max(dp[i-1][j], dp[i][j-1])
Answer: dp[m][n]
```

```cpp
int longestCommonSubsequence(string text1, string text2) {
    int m = text1.length();
    int n = text2.length();
    vector<vector<int>> dp(m+1, vector<int>(n+1, 0));
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1[i-1] == text2[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }
    
    return dp[m][n];
}
```

### 4. 0/1 Knapsack

**Problem:** Maximize value with weight constraint. Each item can be taken at most once.

```
State: dp[i][w] = max value using first i items with capacity w
Base: dp[0][w] = 0 for all w
Transition:
    if w >= weight[i-1]:
        dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i-1]] + value[i-1])
    else:
        dp[i][w] = dp[i-1][w]
Answer: dp[n][W]
```

```cpp
int knapsack(int W, vector<int>& weights, vector<int>& values) {
    int n = weights.size();
    vector<vector<int>> dp(n+1, vector<int>(W+1, 0));
    
    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= W; w++) {
            if (w >= weights[i-1]) {
                dp[i][w] = max(dp[i-1][w], 
                               dp[i-1][w-weights[i-1]] + values[i-1]);
            } else {
                dp[i][w] = dp[i-1][w];
            }
        }
    }
    
    return dp[n][W];
}
```

### 5. Edit Distance (Levenshtein Distance)

**Problem:** Minimum operations (insert, delete, replace) to convert one string to another.

```
State: dp[i][j] = min operations to convert word1[0..i-1] to word2[0..j-1]
Base: dp[0][j] = j, dp[i][0] = i
Transition:
    if word1[i-1] == word2[j-1]:
        dp[i][j] = dp[i-1][j-1]
    else:
        dp[i][j] = 1 + min(dp[i-1][j],     // delete
                           dp[i][j-1],     // insert
                           dp[i-1][j-1])   // replace
Answer: dp[m][n]
```

```cpp
int minDistance(string word1, string word2) {
    int m = word1.length();
    int n = word2.length();
    vector<vector<int>> dp(m+1, vector<int>(n+1, 0));
    
    for (int i = 0; i <= m; i++) {
        dp[i][0] = i;
    }
    for (int j = 0; j <= n; j++) {
        dp[0][j] = j;
    }
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (word1[i-1] == word2[j-1]) {
                dp[i][j] = dp[i-1][j-1];
            } else {
                dp[i][j] = 1 + min(dp[i-1][j], 
                                   min(dp[i][j-1], 
                                       dp[i-1][j-1]));
            }
        }
    }
    
    return dp[m][n];
}
```

### 6. Longest Palindromic Subsequence

**Problem:** Find the length of the longest subsequence that is a palindrome.

```
State: dp[i][j] = length of LPS in s[i..j]
Base: dp[i][i] = 1
Transition:
    if s[i] == s[j]:
        dp[i][j] = dp[i+1][j-1] + 2
    else:
        dp[i][j] = max(dp[i+1][j], dp[i][j-1])
Answer: dp[0][n-1]
```

```cpp
int longestPalindromeSubseq(string s) {
    int n = s.length();
    vector<vector<int>> dp(n, vector<int>(n, 0));
    
    for (int i = 0; i < n; i++) {
        dp[i][i] = 1;
    }
    
    for (int len = 2; len <= n; len++) {
        for (int i = 0; i <= n-len; i++) {
            int j = i + len - 1;
            if (s[i] == s[j]) {
                dp[i][j] = dp[i+1][j-1] + 2;
            } else {
                dp[i][j] = max(dp[i+1][j], dp[i][j-1]);
            }
        }
    }
    
    return dp[0][n-1];
}
```

### Common 2D DP Patterns

| Pattern | Description | Example |
|---------|-------------|---------|
| Grid Path | Moving in grid | Unique Paths, Min Path Sum |
| String Comparison | Comparing two sequences | LCS, Edit Distance |
| Knapsack | Capacity constraint | 0/1 Knapsack, Subset Sum |
| Interval | Subarray/substring problems | LPS, Matrix Chain |
| Palindrome | Palindrome related | Longest Palindromic Substring |

### Space Optimization for 2D DP

Many 2D DP problems only need the previous row.

**Knapsack optimization (2 rows):**

```cpp
int knapsack(int W, vector<int>& weights, vector<int>& values) {
    int n = weights.size();
    vector<vector<int>> dp(2, vector<int>(W+1, 0));
    
    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= W; w++) {
            if (w >= weights[i-1]) {
                dp[i%2][w] = max(dp[(i-1)%2][w], 
                                 dp[(i-1)%2][w-weights[i-1]] + values[i-1]);
            } else {
                dp[i%2][w] = dp[(i-1)%2][w];
            }
        }
    }
    
    return dp[n%2][W];
}
```

**Knapsack optimization (1D array - reversed loop):**

```cpp
int knapsack(int W, vector<int>& weights, vector<int>& values) {
    int n = weights.size();
    vector<int> dp(W+1, 0);
    
    for (int i = 0; i < n; i++) {
        for (int w = W; w >= weights[i]; w--) {
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i]);
        }
    }
    
    return dp[W];
}
```
