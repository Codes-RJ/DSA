# Templates

This folder contains comprehensive starter files and templates for quick problem solving in each language.

## Purpose

- Reduce setup time for competitive programming and DSA practice
- Keep a consistent coding format across different languages
- Make revision and contests faster
- Provide comprehensive implementations of common algorithms and data structures
- Include edge cases and best practices for each language

## Structure

### C++ Templates
- `01_Basic_Template.cpp` - Basic template with utilities, I/O, and common functions
- `02_Algorithm_Templates.cpp` - Comprehensive algorithm implementations
- `03_Data_Structure_Templates.cpp` - Complete data structure implementations

### Java Templates
- `01_Basic_Template.java` - Basic template with utilities and common functions
- `02_Algorithm_Templates.java` - Comprehensive algorithm implementations
- `03_Data_Structure_Templates.java` - Complete data structure implementations

### Python Templates
- `01_Basic_Template.py` - Basic template with utilities and common functions
- `02_Algorithm_Templates.py` - Comprehensive algorithm implementations
- `03_Data_Structure_Templates.py` - Complete data structure implementations

## Features

### Common Features Across All Languages
- Fast I/O for competitive programming
- Mathematical utilities (GCD, LCM, prime checking, etc.)
- String utilities (palindrome check, anagrams, etc.)
- Array utilities (searching, sorting, etc.)
- Common algorithms (Kadane's, two-sum, etc.)

### Algorithm Implementations
- **Sorting Algorithms**: Bubble, Selection, Insertion, Merge, Quick, Heap, Counting, Radix
- **Searching Algorithms**: Linear, Binary, Jump, Interpolation, Exponential
- **Array Algorithms**: Maximum subarray, Two Sum, Three Sum, Missing Number, etc.
- **String Algorithms**: Palindrome, Edit Distance, KMP, Valid Parentheses, etc.
- **Tree Algorithms**: Traversals, BST operations, LCA, etc.
- **Graph Algorithms**: BFS, DFS, Dijkstra, Topological Sort, etc.
- **Dynamic Programming**: Fibonacci, Coin Change, LCS, Knapsack, etc.
- **Bit Manipulation**: Set bits, Power of 2, XOR operations, etc.

### Data Structure Implementations
- **Linear Structures**: Linked Lists, Stacks, Queues
- **Tree Structures**: Binary Trees, BSTs, AVL Trees, Tries
- **Heap Structures**: Min-Heap, Max-Heap
- **Hash Structures**: Hash Tables, Hash Maps
- **Advanced Structures**: Segment Trees, Fenwick Trees, LRU Cache, DSU

## Usage

### For Competitive Programming
1. Copy the appropriate template for your language
2. Modify the `solve()` function for your specific problem
3. Use the provided utilities and algorithms as needed

### For DSA Practice
1. Study the algorithm implementations to understand different approaches
2. Practice with the data structure implementations
3. Use the templates as a reference for best practices

### For Learning
1. Read through the comprehensive comments in each template
2. Understand the time and space complexity analysis
3. Learn the edge cases and optimization techniques

## Best Practices

- Always include input validation and error handling
- Use appropriate data structures for the problem
- Consider time and space complexity
- Handle edge cases properly
- Write clean, readable code with meaningful variable names
- Add comments for complex logic

## Performance Notes

- All templates are optimized for competitive programming
- Fast I/O is implemented for each language
- Memory-efficient implementations are used where possible
- Time complexity is considered for all algorithms
